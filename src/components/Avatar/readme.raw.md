# Avatar

## Props

| Property  | Description                 | Type     | Default        |
| --------- | --------------------------- | -------- | -------------- |
| `alt`     | Alt text for accessibility. | `string` | `undefined`    |
| `picture` | Avatar picture              | `string` | `'avatar.svg'` |
| `size`    | Avatar size                 | `number` | `undefined`    |
